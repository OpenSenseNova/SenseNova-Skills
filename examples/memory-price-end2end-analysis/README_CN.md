# 端到端示例：内存价格分析报告

[English version](README.md)

本示例演示如何把 SenseNova-Skills 中的 **数据分析 → 深度调研 → PPT 生成** 三类技能串联起来，从一份原始 CSV 出发，最终产出一份高端杂志风格的 PPT 汇报。

> 主线：先用数据证明"价格确实在涨、且涨幅集中在关键品类"，再用外部研究解释"为什么涨、背后驱动是什么"，最后给出趋势判断与行动建议。

## 输入

- [`result/芯片价格汇总.csv`](result/芯片价格汇总.csv)：来自 China Flash Market 的存储芯片报价数据，2026-01-22 ~ 2026-04-21，共 1835 条记录，覆盖 5 大类、15 个子类、77 种产品。

## 涉及技能

| 步骤 | 技能 | 作用 |
|------|------|------|
| 1️⃣ 数据分析 | [`sn-da-excel-workflow`](../../skills/sn-da-excel-workflow/SKILL.md) | 读取 CSV、清洗、聚合、产出结构化分析报告 |
| 1️⃣/2️⃣ 渲染 | [`sn-md-to-html-report`](../../skills/sn-md-to-html-report/SKILL.md) | 把 Markdown 报告转成可视化 HTML |
| 2️⃣ 深度调研 | [`sn-deep-research`](../../skills/sn-deep-research/SKILL.md) | 基于数据分析结论扩展外部研究 |
| 3️⃣ PPT 生成 | [`sn-ppt-standard`](../../skills/sn-ppt-standard/SKILL.md)（经 [`sn-ppt-entry`](../../skills/sn-ppt-entry/SKILL.md) 分派） | 把报告内容渲染成 PPTX |

## 操作流程

### 第一步：数据分析

将 `芯片价格汇总.csv` 放到工作目录，向智能体发起：

```text
请读取「芯片价格汇总.csv」，对近期的存储芯片报价数据进行清洗和分析。
```

智能体会触发 `sn-da-excel-workflow`，输出 Markdown 格式的汇总分析报告。

> **数据分析结论**：近期存储价格整体呈上行趋势，其中部分 DRAM 与 NAND 产品涨幅最明显；上涨节奏上，2 月下旬开始出现拐点，3 月后进入加速阶段；不同品类之间分化明显，服务器相关产品表现强于消费类产品，说明本轮上涨并非全面同步，而是由重点品类率先带动。

也可以继续要求把 Markdown 渲染为 HTML，便于浏览器查看：

```text
使用 sn-md-to-html-report 这个 skill 把输出的 markdown 转换成 html 文件
```

### 第二步：深度调研

基于第一步的结论，进一步追问"为什么"：

```text
基于数据分析结果，调研 2026 年以来内存和闪存价格波动的主要原因。
```

智能体会触发 `sn-deep-research`，调用搜索/抓取/综合等子技能，产出图文并茂的深度报告：

- [`result/内存条价格暴涨原因分析/深度调研-内存条价格暴涨原因分析.md`](result/内存条价格暴涨原因分析/深度调研-内存条价格暴涨原因分析.md)
- [`result/内存条价格暴涨原因分析/深度调研-内存条价格暴涨原因分析.html`](result/内存条价格暴涨原因分析/深度调研-内存条价格暴涨原因分析.html)
- [`result/内存条价格暴涨原因分析/figures/`](result/内存条价格暴涨原因分析/figures/)：报告引用的 9 张图表

> **深度调研结论**：本轮价格上涨主要由供给收缩、AI 服务器需求增强以及部分厂商主动控产共同推动；短期看存在情绪和备货带来的波动放大，但中期更像是供需重新平衡下的结构性修复；后续若高端需求持续、原厂延续谨慎供给策略，价格仍有继续上行或高位震荡的可能。

同样可以让智能体把 Markdown 转成 HTML 直接浏览。

### 第三步：生成 PPT

把前两步的报告作为素材，生成最终 PPT：

```text
根据数据分析结果，做一个 ppt 来展示报告内容，要求用到报告中的所有图片，
并且数据和结论要和报告一致，页数控制在 16 页，用标准模式。
整体风格要是高端杂志报告风，颜色以红色金色为主，明亮色背景，要突出重点。
```

智能体经由 `sn-ppt-entry` 进入 `sn-ppt-standard`，输出：

- [`result/半导体存储市场研究报告_20260426_015200.pptx`](result/半导体存储市场研究报告_20260426_015200.pptx)：可直接打开的 PPTX 文件
- [`result/半导体存储市场研究报告_20260426_015200.zip`](result/半导体存储市场研究报告_20260426_015200.zip)：PPT 对应的原始分页 HTML 集合，便于二次编辑或直接网页预览
